<h3 align="center">Smart-Calculator</h3>

# 📌 Overview

Smart Calculator is a modern and responsive web-based calculator developed using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division through an intuitive and interactive interface. The project focuses on clean UI design, smooth user interactions, and real-time calculation results, making it suitable for both everyday use and frontend learning practice.

# 🔧 Technologies Used

HTML

CSS

JavaScript
